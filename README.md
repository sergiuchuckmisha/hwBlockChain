# hwBlockChain
Реализовать приложение которое умеет:
   - читать введенные данные и записывать из в БД
   - создавать структуру типа BlockChain когда каждая транзакция (под транзакцией понимаются данные плюс служебная информация (время добавления)) ссылается на предыдущую посредством хэширования
   
   Используемые технологии:
   - БД: MySQL 
   - ORM: Hibernate
   - Spring-MVC
   - Spring-IoC
   - pool of connections: c3p0
